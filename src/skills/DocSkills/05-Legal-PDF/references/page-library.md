# Page Library — Legal Documents (816×1056px Portrait)

Seven page patterns for legal documents.

---

## Pattern Index

| # | Name | Best For |
|---|---|---|
| L1 | Title + Parties | Document title, party identification, effective date |
| L2 | Recitals + Definitions | WHEREAS clauses, defined terms |
| L3 | Clause Body | Standard articles with sections and sub-clauses |
| L4 | Table of Obligations | Structured obligations, schedules, payment terms |
| L5 | General Provisions | Boilerplate: entire agreement, severability, waiver, notices |
| L6 | Signature Page | IN WITNESS WHEREOF, signature blocks, witness lines |
| L7 | Schedule / Exhibit | Attached schedules, exhibits, appendices |

---

## Document Type → Pattern Mapping

### NDA (Non-Disclosure Agreement)
| Page | Content | Pattern |
|---|---|---|
| 1 | Title, parties, date | L1 |
| 1–2 | Recitals, definitions | L2 |
| 2–3 | Obligations, exclusions, term | L3 |
| 4 | General provisions | L5 |
| 4–5 | Signatures + AI disclaimer | L6 |

### Service Agreement
| Page | Content | Pattern |
|---|---|---|
| 1 | Title, parties, date | L1 |
| 1–2 | Recitals, definitions | L2 |
| 2–4 | Scope, compensation, term, IP, confidentiality | L3 (multiple pages) |
| 4–5 | Payment schedule (if applicable) | L4 |
| 5–6 | Limitation of liability, indemnification, general provisions | L3 + L5 |
| 6–7 | Signatures + AI disclaimer | L6 |
| 7+ | SOW / exhibits | L7 |

### Terms of Service
| Page | Content | Pattern |
|---|---|---|
| 1 | Title, effective date | L1 (simplified — no party blocks) |
| 1–5 | All clauses (acceptance through governing law) | L3 (multiple pages) |
| 5 | Contact information | L5 |

### Privacy Policy
| Page | Content | Pattern |
|---|---|---|
| 1 | Title, effective date | L1 (simplified) |
| 1–4 | All sections | L3 (multiple pages) |
| 4 | Contact, changes | L5 |

### MOU
| Page | Content | Pattern |
|---|---|---|
| 1 | Title, parties | L1 |
| 1 | Purpose | L2 |
| 2 | Scope, roles, duration | L3 |
| 3 | Confidentiality, termination, non-binding | L5 |
| 3 | Signatures | L6 |

---

## Variety Rules

- L1 always first
- L6 always last (before any L7 exhibits)
- L3 can repeat for multi-article documents (most documents are primarily L3)
- L2 appears only once (definitions are consolidated)
- L5 appears only once (general provisions)
- L4 is optional — only when structured schedules/tables are needed
- Legal documents are inherently repetitive in layout — L3 dominance is expected and correct
